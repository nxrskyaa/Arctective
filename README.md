# ARCTECTIVE: THE FINAL RECEIPT

A retro arcade pixel-noir detective game for Arc Testnet.

In Arc City, every payment leaves a clue. Every receipt has a shadow. Every
solved case becomes onchain reputation.

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- wagmi + viem
- Zustand-ready/localStorage gameplay state

## Arc Testnet

- Chain ID: `5042002`
- RPC: `https://rpc.testnet.arc.network`
- Explorer: `https://testnet.arcscan.app`
- Native gas: `USDC`
- Casebook contract: `0x6b3BaF2Ed389D23fa30203Dc1ef6AEa288f51995`
